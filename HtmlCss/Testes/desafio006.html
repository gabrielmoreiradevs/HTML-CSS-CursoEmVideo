Tags em HTML

As tags em HTML são usadas para estruturar e formatar o conteúdo de uma página web. Elas são delimitadas por colchetes angulares (< >) e geralmente vêm em pares, com uma tag de abertura e uma tag de fechamento.

  • Parágrafos
  • Títulos
  • Imagens
  • Abreviações
  • Códigos-Fonte
  • Listas Numeradas
  • Listas com Demarcadores
